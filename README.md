# 🌍 Rotaly

> Akıllı Seyahat Bütçe ve Planlama Platformu

Rotaly, seyahate çıkmadan önce tüm maliyetleri tek platformda planlamayı sağlayan modern bir web uygulamasıdır.

## ✨ Özellikler

- 📍 Başlangıç ve varış şehri seçimi
- 🚗 Ulaşım türü seçimi
- 👥 Kişi sayısı
- 📅 Tatil süresi
- ⛽ Yakıt maliyeti hesaplama
- 🛣️ HGS / otoyol ücretleri
- 🏨 Konaklama bütçesi
- 🍽️ Yemek bütçesi
- 📊 Toplam seyahat maliyeti

## 🛠️ Teknolojiler

- Next.js
- React
- TypeScript
- Tailwind CSS
